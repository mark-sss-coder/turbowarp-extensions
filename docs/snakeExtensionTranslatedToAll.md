# Snake
A big extension generated by AI, tested, and have so much possible modes.
## Blocks

### `Game Board Text` (Reporter)
Returns the entire rendering grid as a text string filled with custom emojis. You should continuously map this reporter to a Scratch stage monitor or variable.

### `Start: Width [W] Height [H] Mode [MODE]` (Command)
Initializes a new game session with a custom grid resolution and configures the active rule set.
- **W / H**: Grid dimensions (minimum 5x5).
- **MODE**: The ID or dynamic text mixin formula for the current round.

### `Setup: Speed (ms) [SPD] Start Length [LEN]` (Command)
Configures the mechanical core of the game.
- **SPD**: Tick rate in milliseconds (lower means faster gameplay).
- **LEN**: The starting length of the snake.

### `Skins: GridA [A] GridB [B] Apple [APL] Head [HD] Body [BD] BrokenShield [SHD]` (Command)
Allows full emoji skin customization for the game grid and entities.

### `Game Action: [ACTION]` (Command)
Controls the current game state (`start`, `pause`, or `reset`).

### `Turn Snake: [DIR]` (Command)
Changes the movement vector of the snake's head (`up`, `down`, `left`, `right`).

### `Current Score` (Reporter)
Returns the total number of fruits consumed during the active round.

The core feature of this engine is the ability to **combine multiple modes simultaneously** without any internal script conflicts. To create a custom gameplay loop, combine mode numbers using the **plus sign (`+`)** inside a green Scratch `join` operator.
